# Futbol Stats Web Application

## Overview
Futbol Stats is an interactive web application that displays football match statistics and summaries. Users can search for clubs or leagues and dynamically view match details.

## Features
- Dynamic match selection
- Search functionality
- Interactive match summary display
- Responsive UI design
- Structured statistics table rendering

## Technologies Used
- HTML
- CSS
- JavaScript
